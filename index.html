pwd (the directory when I am)

ls (list of files)

cd (change directory)

cd~ (come back the initial directory)

touch xxx (create file)

code . (open visual studio code in WSL ubuntu)

sudo (gift administrator permission)

apt-get update (download updates, it always is with sudo)

apt-get upgrade (run updates, it always is with sudo)

apt install nodejs or xxx (install some new, it always is with sudo)
